## Strongest judge path

1. Watch the 4:49 final demo from start to finish.
2. Open the live product and inspect the World Cup-only catalog and provenance labels.
3. Start the complete replay and confirm scoreboard, momentum, timeline, controls and Match Brief evolve together.
4. Open Fan Zone and verify the wallet-owned identity, check-in, quiz and 36-item reward loop.
5. Open fixture chat and explain fresh Phantom message signing versus fee-paying transactions.
6. Run Judge Room and wait for all eight production checks to pass.
7. Open the public devnet program, one successful receipt and the retired-index rejection in Explorer.
8. Open GitHub Actions and the final technical report.

## Why the submission is differentiated

- A late fan shares exactly what they have watched, not a URL that can leak the ending.
- Replay uses the same product state as live coverage rather than a separate static result page.
- Solana is used for ownership, receipts and replay resistance, not added as an unrelated token layer.
- Fan progression is non-financial and non-transferable.
- Chat contains no seeded users and ties public identity to a wallet-owned alias.
- Data gaps stay visibly unavailable; the product does not invent competitions, kick-off times or knockout winners.
- The final video shows real product behavior, Phantom approvals and immediate transaction receipts.

## Residual risks

1. Live TxLINE access is licensed only through the published hackathon window; written permission is needed for continued live operation.
2. Chat presence/history use bounded single-instance memory; scale requires managed pub/sub, persistence and moderator tooling.
3. Closed-app push notifications need a complete Web Push subscription/revocation service.
4. The commercial model is technically credible but has no signed customer pilot yet.
5. Competitor quality and judge preferences are not public, so top-three probability cannot be stated responsibly.

## Submission decision

The product is ready to submit. Remaining work is operational rather than engineering: YouTube upload, optional X post, private-window link check, eligibility confirmation and personal Superteam submission before the deadline.
